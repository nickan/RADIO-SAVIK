# Радио Савик — Инструкция для магазинов

## Что в папке

| Файл | Назначение |
|------|------------|
| `ЗАПУСК.bat` | **Двойной клик** — открывает радио в браузере |
| `RADIO.html` | Веб-страница радио (можно открыть вручную) |
| `script.js` | Скрипт (не трогать) |
| `file/` | **Папка с рекламой** — сюда положить MP3 файлы |

## Как пользоваться

1. **Положите MP3 файлы рекламы** в папку `file/`
2. **Запустите** `ЗАПУСК.bat` (или откройте `RADIO.html` в Chrome/Edge)
3. Выберите радиостанцию (по умолчанию — Radio Record)
4. Нажмите **▶ Старт**
5. Выберите папку `file/` когда появится диалог
6. Готово! Радио играет, реклама вставляется автоматически

## Настройки (⚙️)

Нажмите **шестерёнку** справа от названия:
- **Интервал рекламы** — как часто вставлять рекламу (в минутах)
- **Громкость радио** — отдельная регулировка
- **Громкость рекламы** — отдельная регулировка

Все настройки **сохраняются** автоматически.
