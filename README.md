# 📻 РАДИО САВИК (RADIO SAVIK)

**РАДИО САВИК** — это стильный неоновый веб-плеер, специально разработанный для использования в магазинах и общественных пространствах. Он позволяет транслировать онлайн-радио и автоматически вставлять локальную рекламу через заданные промежутки времени с плавными аудио-переходами.

## ✨ Основные возможности

*   **🎧 Умная микшировка**: Радио плавно затихает (**fade-out**) перед началом рекламы и плавно возвращается (**fade-in**) после её завершения.
*   **⏱️ Гибкий интервал**: Настройка времени между рекламными блоками в минутах.
*   **🔊 Раздельный звук**: Отдельные регуляторы громкости для радиопотока и для рекламных роликов.
*   **📂 Drag & Drop**: Просто перетащите аудиофайлы (.mp3, .wav и др.) прямо в окно браузера для формирования списка рекламы.
*   **⚙️ Сохранение настроек**: Все параметры (URL радио, громкость, интервал) автоматически сохраняются в `localStorage` вашего браузера.
*   **📊 Таймер обратного отсчета**: Визуальный счетчик времени до следующего рекламного выхода.
*   **⏸️ Пауза с сохранением состояния**: Возможность приостановить всё воспроизведение без сброса таймера рекламы.

## 🚀 Как запустить

### Для Windows (в один клик):
Дважды щелкните по файлу **`ЗАПУСК.bat`**. Это автоматически откроет приложение в вашем браузере по умолчанию.

### Вручную:
Просто откройте файл `RADIO.html` в браузере (рекомендуются **Google Chrome** или **Microsoft Edge**).

## 🛠️ Настройка для магазинов

1.  **Загрузка рекламы**: Положите MP3-файлы в папку `file/`. При нажатии кнопки "Старт" плеер предложит выбрать эту папку (из-за ограничений безопасности браузера выбор папки делается один раз вручную).
2.  **Выбор радио**: По умолчанию стоит **Radio Record**. Вы можете вставить ссылку на любой аудиопоток (Icecast/Shoutcast) в поле "Радио поток".
3.  **Панель настроек**: Нажмите на иконку шестерёнки **⚙️** справа от названия, чтобы настроить индивидуальные уровни громкости и частоту выхода рекламы.

## 📁 Структура проекта

*   `RADIO.html` — Оболочка приложения с неоновым UI и Canvas-фоном.
*   `script.js` — Логика управления звуком, таймерами, анимацией и Drag&Drop.
*   `ЗАПУСК.bat` — Лаунчер для быстрого старта в Windows.
*   `ИНСТРУКЦИЯ.md` — Подробное руководство для конечных пользователей.
*   `file/` — Папка для размещения рекламных материалов.
